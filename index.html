Why Animate?

ANIMATE 101:

Animation is a fun and creative way to bring characters and stories to life! In this tutorial, I will be showing you how to create a pixel art character and animate it! We will be creating Calcifer from Howl's Moving Castle. This tutorial is beginner friendly and I will be using free software called Pixilart. Let's get started!

QUICK TIP:

I use Pixilart, Pixquare, and Procreate. Pixilart and Pixquare are free (I use both) and I use Procreate too (around $12). Aesprite is another paid software (around $20) that is popular for pixel art but I don't use it cause I'm cheap! Let's use Pixilart for this tutorial. Start a canvas (I do 64px x 64px) and let's get started!

PROCESS

STEP 1:

Use the circle tool to create a circle! It makes drawing easier.

QUICK TIP:

When using the circle tool, you can drag and hold shift to make a perfect circle! This works for squares too!

STEP 2:

I know it looks creepy, but we will make edits later on! Draw the eyes and the mouth of Calcifer!

QUICK TIP:

Use your reference image to help you draw the eyes and mouth! It helps a lot when drawing something new!

STEP 3:

Draw Calcifer's body and flames to complete the character!

QUICK TIP:

To make things faster, use the mirror x and or y function when drawing symmetrical things! It will automatically draw the other side for you.

STEP 4:

Fixing up the eyes and mouth to make it look cuter! Adding shading and highlights to make it pop more! Add in the color of Calcifer.

QUICK TIP:

Use your reference image to help you draw the eyes and mouth! It helps a lot when drawing something new!

STEP 5:

Color in his body and make sure to use a reference.

QUICK TIP:

To make things efficient, use the fill/bucket tool to color in large areas! It saves a lot of time.

STEP 6:

Create a gradient for the body and flames to make it look more dynamic! Add in the finishing touches and details to complete the design!

QUICK TIP:

Instead of using black as an outline color, I use a dark brown/red color to make it look softer!

STEP 7:

Onto the animation portion, start by creating keyframes for the main movements. We want to make it look like Calcifer is flickering, so every frame will be different from the previous. We want to increase the flame, so coloring the pixels outwards (like he's growing) and then back inwards (like he's shrinking). This is the final step! The next step is optional.

QUICK TIP:

To make things efficient, I use copy and paste to duplicate frames and then just edit them slightly.

STEP 8:

OPTIONAL: Add dynamic flames around Calcifer to make it look more interesting! Make sure to use colors that match the character. Add in the finishing touches and details to complete the animation!

QUICK TIP:

Instead of using black as an outline color, I use a dark brown/red color to make it look softer!
